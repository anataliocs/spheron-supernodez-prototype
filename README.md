## Tech Stack

- [Next.js](https://nextjs.org/docs)
- [Tailwind CSS](https://tailwindcss.com/)
- []()

## Run Locally

Install dependencies

```bash
  yarn
```

### Run the Project

```bash
  yarn dev
```
Go to http://localhost:3000

### 2. Environment Variables
Create environment variables file
```bash
  touch .env
```

### Spheron Access Token

Use this `.env` config:
